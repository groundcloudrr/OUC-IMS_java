# Java基本数据类型、类、循环和分支

## 实验目的

本实验旨在综合练习Java的基本数据类型、类、循环和分支结构，通过实现一个简单的学生成绩管理系统，巩固面向对象编程基础和控制流程的应用。

## 实验要求

1. 使用Java基本数据类型和数组进行数据存储
2. 定义类封装学生信息和成绩数据
3. 运用分支结构实现不同功能选择
4. 使用循环结构进行数据遍历和重复操作
5. 实现基本的成绩分析功能

## 实验内容

### 学生类 (Student) 设计

定义一个 Student 类，包含以下属性和方法：
- 属性：学号(id，字符串类型)、姓名(name，字符串类型)、java成绩(javaScore，整型)、数学成绩(mathScore，整型)
- 方法：构造方法、获取总成绩方法、获取平均成绩方法、显示学生信息方法

### 成绩管理类 (GradeManager) 设计

实现主要功能方法：
1. 添加学生信息（添加输入验证，确保成绩在0-100范围内）
2. 显示所有学生信息
3. 根据学号查找学生
4. 计算全班平均分
5. 统计成绩等级分布（A:90-100, B:80-89, C:70-79, D:60-69, E:0-59）
6. 查找最高分和最低分学生

### 主程序流程
使用循环和分支结构实现用户交互菜单：

```
请选择功能：
1.添加学生
2.显示所有学生信息
3.按学号查找学生
4.计算全班平均分
5.统计成绩分布
6.查找最高分和最低分
0.退出系统
```
